# TesisatMarketMobile

Bu proje, tesisat malzemeleri satan bir mobil uygulama için geliştirilmiştir.

## Başlangıç

1. Bağımlılıkları yükleyin
   ```bash
   npm install
   ```

2. Uygulamayı başlatın
   ```bash
   npx expo start
   ```

## Özellikler

- Ana Sayfa
  - Banner/Slider
  - Popüler Kategoriler
  - Öne Çıkan Ürünler

- Kategoriler
  - Su Tesisatı
  - Elektrik
  - Hırdavat
  - Isıtma

- Sepet
  - Ürün Ekleme/Çıkarma
  - Toplam Fiyat Hesaplama

- Profil
  - Kullanıcı Bilgileri
  - Sipariş Geçmişi
  - Adres Yönetimi

## Teknolojiler

- React Native
- Expo
- TypeScript
- React Navigation
